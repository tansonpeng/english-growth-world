# 英语成长世界

一个面向学前儿童的英语启蒙网页，采用地图式首页、动物引导和极简学习流程，适合手机、平板和电脑浏览。

## 在线地址

- GitHub Pages: [https://tansonpeng.github.io/english-growth-world/](https://tansonpeng.github.io/english-growth-world/)

## 产品特点

- 地图优先：首页先展示世界地图和动物节点，不依赖文字说明
- 低龄友好：主交互是点击、听音、跟读，减少阅读负担
- 动效引导：云朵、路径和角色都有轻量动画，帮助孩子建立注意力锚点
- 家长入口独立：家长信息单独收纳，不干扰孩子主流程
- 响应式布局：支持手机、平板、电脑不同屏幕尺寸

## 本地运行

直接用任意静态服务器打开即可，例如：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 目录说明

- `index.html` 页面结构
- `styles.css` 视觉样式和响应式布局
- `app.js` 动画、语音和页面交互
- `audio/` 语音素材

## 交互说明

- 点击地图上的动物节点，会进入极简学习页
- 点击 `🔊` 可以朗读当前单词
- 点击 `开始` 会触发语音提示
- 点击右上角 `家长` 可以查看简单学习数据

## 部署说明

当前版本已部署到 GitHub Pages。后续如果更新代码，只需要重新提交到 `main` 分支，Pages 会自动重新构建。

## 适配建议

- 手机：优先保证地图和动物节点的可点击区域足够大
- 平板：保留地图主视觉，允许更完整地显示顶部栏
- 电脑：充分利用横向空间，让地图保持“探索感”
